# Tutorial de Angular: Gestor de Bookmarks

Actividad 2 de Programación de Aplicaciones en Red del Grado de Ingeniería Informática de la UNIR.

Este es un tutorial interactivo de Angular donde la propia aplicación enseña cómo está construida. En lugar de documentación separada, la aplicación "se explica a sí misma": al hacer clic en cualquier componente, un panel lateral muestra su código y los conceptos de Angular que utiliza.

La aplicación en sí es un gestor de bookmarks funcional. Puedes añadir, filtrar y eliminar marcadores mientras aprendes cómo funciona cada pieza.
